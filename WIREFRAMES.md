   - Click to instantly add sub-function
   - Button fades out after clicking

   - Collapsible/expandable
   - Color-coded status:

### Transitions
- Page transitions: 200ms ease-in-out
- Card hover: 150ms ease
- Dropdown expand: 300ms ease-out
- Modal open: 250ms ease-out

- Buttons: Background darkens 10%, scale 102%
- Cards: Border color intensifies, shadow increases
- Links: Underline appears, color darkens

## Accessibility Notes

1. **Keyboard Navigation**
   - Tab order follows visual flow
   - Enter to select/activate
   - Escape to close modals
   - Arrow keys for radio groups

2. **Screen Readers**
   - All icons have aria-labels
   - Form fields have proper labels
   - Status messages announced
   - Progress indicator readable

3. **Color Contrast**
   - All text meets WCAG AA standards
   - Status indicators have icons + color
   - Focus indicators clearly visible

4. **Error Handling**
   - Inline validation messages
   - Error summary at top of form
   - Clear actionable guidance
